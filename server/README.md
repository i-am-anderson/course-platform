# API Simples

# Funcionamento

- Solicita uma questão pelo ID da questão

```bash
  GET http://localhost:5000/api/v1/question/1
```

- Verifica a resposta pelo ID da questão e pelo ID da resposta

```bash
GET http://localhost:5000/api/v1/question/1/answer/1
```
